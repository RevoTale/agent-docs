# Awesome Registry
This registry defines enforced utilities and libraries by stack.

Rules:
- Read the matching stack awesome file before introducing, replacing, or removing libraries.
- Use `required` entries for matching capabilities.
- Any deviation requires explicit user `Accept` and must be documented as a strict-rule exception.

| stack_key | awesome_file | scope |
| --- | --- | --- |
| go | [go.md](./go.md) | Go runtime, backend services, Go templating, and Go GraphQL client |
| react | [react.md](./react.md) | React repositories and applications |
| nextjs | [nextjs.md](./nextjs.md) | Next.js repositories and applications |
